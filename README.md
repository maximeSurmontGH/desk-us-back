## Description

Desks Us is a to do app working with rooms.
A Room regroup major tasks splittable in minor tasks and a global room chat.

Back end repository.

Based on [Nest](https://github.com/nestjs/nest) framework starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
